# TagSelector

<!-- STORY -->

## Introduction

TagSelector is an easy-to-use component to select tags for blogs.

## Usage

```javascript
import { TagSelector } from "some-package-name";
```

## Example use

```javascript
const myPage = (props) => {
	return (
		<main>
			<TagSelector />
		</main>
	);
};
```

## Properties

It uses Categories object from Categories.js
