# UserCard

<!-- STORY -->

## Introduction

UserCard is an easy-to-use component.

## Usage

```javascript
import { UserCard } from "some-package-name";
```

## Example use

```javascript
const myPage = (props) => {
	return (
		<main>
			<UserCard />
		</main>
	);
};
```

## Properties

- `id` - is the author id, whose value value will be fetched

| propName | propType | defaultValue | isRequired |
| -------- | -------- | ------------ | ---------- |
| id       | string   | -            | Yes        |
